Python Comments Last updated: 2019-06-01

Python code can be explained using comments. Comments can be used to improve the readability of the code. When testing code, comments can be used to prevent execution.

Writing a Comment

Python will ignore comments that begin with a #. Python will ignore the rest of the line if a comment is placed at the end of a line. A comment does not have to contain text that describes the code; it can also be used to block Python from running code.

Python doesn't have a syntax for multi-line comments. To add a multiline comment we use a # for each line.
